## Minimum Viable Product
Lunchin is a Yelp clone optimized for food trucks.

Users can
- [ ] See what food trucks are near them
- [ ] Search
- [ ] Create accounts
- [ ] Log in
- [ ] Give ratings/reviews
- [ ] Read reviews
Business Owners can
- [ ] Update their location
- [ ] Post their schedule
